* 概要
  水惑星設定(陸・地形なし)の静止した海洋に対して, 東西一様な海面応力場を与え, 
  どのような循環が駆動されるかをシミュレーションする. 

  地球では偏西風や貿易風の海面応力によって, エクマン層を介して循環が駆動される. 
  
* 実験設定

** 各実験の設定の違い

  | 実験名 | 海面応力場                    | 解像度　 |   |
  |--------+-------------------------------+----------+---|
  | exp0   | 簡単な解析関数で表す          | T21L20   |   |
  | exp1   | Marshall, /et al/ (2007) の Fig.3 | T21L20   |   |
  | exp2   | Marshall, /et al/ (2007) の Fig.3 | T21L30   |   |


** 各実験で共通な設定

- 計算領域について

  計算領域は, 全球的に広がる水深 5.3 km の海洋である. 
  海底地形や陸は存在しない.  

   
　
* exp0: 理想的な海面応力場を与えた場合の結果
  

* exp1: Marshall et al(2007) の Fig.3 の海面応力場を与えた場合
